# Reference

Auto-generated API references, one per language plus REST.

| Language / surface | Generator | Source |
|--------------------|-----------|--------|
| [Python](python/index.md) | mkdocstrings | `platform/{jobsvc,calibration,launcher}/src`, `spinor/submit/python/spinor_submit` |
| [C++](cpp/index.md) | Doxygen (HTML embed) | `spinor/`, `phonon/`, `photon/` public headers |

| [REST](rest/index.html) | Redoc | `jobsvc` OpenAPI export |

These pages are regenerated on every push to `main` by the docs CI.
The **handwritten** narrative for each language lives one section
up, under [SDKs](../sdks/index.md).

## How to read each surface

- **Python** — every public symbol with parameters, returns,
  raises, and examples. Generated by `mkdocstrings-python` from
  Google-style docstrings on each module.
- **C++** — Doxygen. Use the search box; class hierarchies are
  under "Classes", function indexes under "Files".
- **TypeScript** — TypeDoc. Each export has its full type
  signature inline.
- **REST** — Redoc. One page per endpoint, with try-it-out forms
  if the OpenAPI server URL is configured.

## Stability

| Surface | Stable across | Caveat |
|---------|---------------|--------|
| Python | minor versions (0.x) | symbols starting with `_` are private |
| C++ | major versions only | ABI is not yet frozen; pin major and rebuild on minor bumps |
| TypeScript | minor versions (0.x) | types regenerate from OpenAPI; transitively stable when the REST API is |
| REST | minor versions (0.x) | OpenAPI changes are documented in the [changelog](../changelog.md) |
